PostgreSQL (x.x = 9.1) and PostGIS
    -sudo apt-get install postgresql-x.x
    -sudo apt-get install postgresql-x.x-postgis
    -sudo apt-get install postgresql-server-dev-x.x
    -sudo apt-get install python-psycopg2

    To properly install PostGIS 2.0+:
        -sudo apt-get install python-software-properties
        -sudo apt-add-repository ppa:ubuntugis/ppa
        -sudo apt-get update
        -sudo apt-get install postgresql-9.1-postgis

    Switch to postgres user, create a PostgreSQl database, open it in the psql terminal and add the postgis extension
        $ su postgres
        $ createdb <db name>
        $ psql <db name>
        postgres=# CREATE EXTENSION postgis;

    Then create a *database* account named postgres to match the system user account:
        $ su postgres
        postgres=# \password postgres
        Enter new password: 
        Enter it again: 
        postgres=# \q
        postgres$

    In Django settings.py, set user and pwd to match this user/pwd
        'ENGINE': 'django.contrib.gis.db.backends.postgis',
        'NAME':'<db name>',
        'USER':'postgres',
        'PASSWORD':'<postgres pwd>',
